# Product Overview

Take Your Time (TYT) is a VS Code extension that provides developers with instant access to web-based arcade games while waiting for AI agents or long-running tasks to complete.

## Core Functionality

- Activity bar icon for quick access to games
- Embedded game emulator in editor tabs
- Multiple simultaneous game tabs
- Non-blocking operation that doesn't interfere with background tasks
- Seamless integration with VS Code's editor experience

## Target Users

Developers who want to make productive use of waiting time during:
- AI agent execution
- Long-running builds or tests
- Code compilation
- Deployment processes

## Key Value Proposition

Transforms idle waiting time into enjoyable breaks without leaving the IDE, helping developers maintain focus and reduce context switching.
