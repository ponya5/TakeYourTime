# Take Your Time (TYT)

A VS Code extension that provides developers with instant access to web-based arcade games while waiting for AI agents or long-running tasks to complete.

## Features

- **Activity Bar Icon**: Quick access to games.
- **Single-Click Launch**: Opens a game in a new tab immediately.
- **Safe & Secure**: Runs in a sandboxed webview.

## Extension Settings

This extension contributes the following settings:

* `takeYourTime.gameUrl`: URL of the game emulator site (default: https://www.smbgames.be/)
* `takeYourTime.fallbackUrl`: Fallback URL if the primary site is unavailable.
* `takeYourTime.enableErrorReporting`: Enable/disable error reporting.
